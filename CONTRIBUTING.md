# Contributing

Thanks for contributing to Sonata SDK!

## Development

```bash
git clone https://github.com/sonata-sdk/sonata-sdk-packages.git
cd sonata-sdk-packages/packages/plugin-sdk
npm install
npm run build
```

## Adding a new package

1. Create `packages/<name>/` with `package.json` scoped to `@sonata-sdk/`
2. Add it to the root `README.md` table
3. Open a pull request

## Pull requests

- Keep changes focused
- Run `npm run build` before submitting
- Update docs if needed
